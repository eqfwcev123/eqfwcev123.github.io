우분투에서 도커 설치/삭제

Posted on 2020-01-29 In Docker

우분투에 Docker 설치하기

sudo apt-get remove docker docker-engine docker.io containerd runc : 이전 Docker 버전 삭제

Docker-CE 를 처음 설치할 경우 Docker repository 를 설정 해줘야 한다. 레포를 설정 한 이후 Docker 를 설치/업데이트 할 수 있다.

  1. sudo apt-get update : apt 패키지 업데이트

  2. sudo apt-get install \
    apt-transport-https \
    ca-certificates \
    curl \
    gnupg-agent \
    software-properties-common

# apt가 HTTPS를 통해 저장소를 사용할 수 있도록 패키지를 설치해준다

Docker 설치

  1. sudo apt-get update
  2. sudo apt-get install docker-ce docker-ce-cli containerd.io
  3. sudo docker run hello-world : docker 가 설치 되었는지 확인하기

Docker 제거

sudo apt-get purge docker-ce : Docker CE 를 삭제한다

sudo rm -rf /var/lib/docker: Docker 의 이미지, 컨테이너 등은 자동으로 삭제가 되지 않기 때문에 직접 삭제를 해줘야 한다.
