django-secrets 사용 방법

Posted on 2020-02-01 In Server Deploy

Django Secrets Manager

django secrets manager 는 사용자의 secret를 관리해주는 패키지이다.

django secrets manager를 사용하기 위해서는 다음 settings를 설정 해줘야 한다

  • AWS_SECRETS_MANAGER_SECRET_NAME : secret 이름 혹은 SecretsManager

  • AWS_SECRETS_MANAGER_SECRET_SECTION: Json 파일에서 secret value의 키값. 즉, Json의 키값

    • instagram:{
    AWS_ACCESS_KEY_ID: "....",
    AWS_SECRET_ACCESS_KEY: "..."
}

AWS Credentials 설정

Django 는 두가지 방법으로 AWS SecretsManager에 접근할 수 있다.

  1. ~/.aws/credentials 파일
  2. 환경 변수

boto3에서 사용하는 방법과 마찬가지로 credentials 파일에 SecretsManager에 AWS SecretsManager의 id 와 access key를 저장

두번째 방법으로 환경변수에 다음 환경 변수들을 작성:

AWS_SECRETS_MANAGER_ACCESS_KEY_ID

AWS_SECRETS_MANAGER_SECRET_ACCESS_KEY
