Understanding Process, Thread and Data structures

Process A program in execution is often referred as process and a thread is a subset of a process. A process can either contain one thread (Single thread process) or mutiple threads(Multi-thread process). Inside a Process contains 4 different memory area: Data, Code, Stack, Heap Thread A unit of work that runs independently within a process. Specifically, a unit of work or task managed by the operating system. Threads and tasks (or tasks) can be used interchangeably. A single process is occured by JVM and statements within main()are a single thread. To create another thread other than main (), either inherit the Thread class or implement the Runnable interface. In multi-threaded operation, each thread can send and receive information to reduce errors in processing. Processes cannot send or receive information with each other.
