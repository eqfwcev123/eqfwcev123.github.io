JavaScript Prototype

JavaScript StudyPrototypeHow porototype works in JavaScript: 12345678910function Person(name){ this.name = name;}Person.prototype.isHuman = true;let person1 = new Person('Lee')let person1 = new Person('john')console.log(person1.__proto__.isHuman) //trueconsole.log(person2.__proto__.isHuman) //true A Person constructor function is created during function declaration (Syntax analysis stage) and when the function is created, the prototype of the function is also created. Rather than storing isHuman inside the function, the isHuman property is stored inside Person.prototype and is shared by all its instances.To create an instance of Person constructor function, the new keyword is used and 'Lee','Kim' are passed as arguments to the constructor function. Before using the new keyword Javascript engine always checks whether the function is a constructor or not.
